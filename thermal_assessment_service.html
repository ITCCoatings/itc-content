3. What the Assessment Includes

On-Site (Greg + Thermal Camera)

  1. Infrared shell survey — Walk the perimeter. Capture thermal images at regular intervals and at any visible hot spots.
  2. Temperature mapping — Record surface temperatures at key measurement points. Document ambient conditions.
  3. Visual inspection — Note shell condition, existing insulation, repair history, access points.
  4. Operating data collection — Current fuel consumption, operating temperature, cycle times, refractory age, maintenance frequency.

Engineering Analysis (Greg + HeatFlow)

  1. Heat flow calculations — Model current heat loss using ITC's proprietary HeatFlow tool. Account for construction type, materials, and operating conditions.
  2. Coating simulation — Model the same furnace with ITC coating applied. Calculate delta in BTU loss, shell temperature reduction, and refractory stress reduction.
  3. ROI projection — Translate BTU savings into dollars using current fuel costs. Factor in extended refractory life (2–5x), reduced maintenance, and potential production gains.

Deliverable

  1. ITC Thermal Assessment Report — Branded PDF containing:
    • Executive summary with headline savings number
    • Thermal images with temperature annotations
    • Heat loss calculations (before and after)
    • Projected annual savings
    • Recommended ITC coating system
    • Relevant case studies
    • Scope of work and budgetary estimate
